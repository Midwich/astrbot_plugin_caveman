# 🪨 astrbot_plugin_caveman — Плагин сжатия выходных токенов LLM

**Версия**: v0.0.1 | **Автор**: Midwich | **Лицензия**: MIT

> *зачем много токен, когда мало токен тоже работать*

Плагин для [AstrBot](https://github.com/AstrBotDevs/AstrBot), который заставляет AI-ассистента отвечать максимально лаконично — экономия **~65% выходных токенов** при полной сохранении технической точности.

Адаптация [**JuliusBrussee/caveman**](https://github.com/JuliusBrussee/caveman) (v1.8.1) — вирусного скилла для Claude Code.

---

## Возможности

- **Четыре уровня сжатия**: lite → full → ultra → wenyan (классический китайский)
- **Состояние на уровне сессии**: каждая сессия чата запоминает свой режим caveman
- **Поддержка i18n**: пользовательский текст адаптируется под `bot_language` AstrBot
- **Нулевые зависимости**: чистый AstrBot API, без дополнительных пакетов
- **Безопасная инъекция**: защита от дублирования промпта через маркер `[CAVEMAN_MODE_ACTIVE]`

## Быстрый старт

### Установка

1. Клонируйте в папку плагинов AstrBot:

```bash
cd /path/to/astrbot/data/plugins
git clone https://github.com/Midwich/astrbot_plugin_caveman.git
```

2. Перезапустите AstrBot или перезагрузите плагины через WebUI.

### Требования

- AstrBot ≥ 3.0 (использует хук `filter.on_llm_request`)

### Настройка

Ручная настройка не требуется. Плагин автоматически читает `bot_language` из настроек AstrBot.

Поддерживаемые языки:

| Язык | Файл | Статус |
|------|------|--------|
| **English** | `.astrbot-plugin/i18n/en-US.json` | ✅ Полный |
| **Русский** | `.astrbot-plugin/i18n/ru-RU.json` | ✅ Полный |
| **中文** | `.astrbot-plugin/i18n/zh-CN.json` | ✅ Полный (фолбек) |

## Команды

| Команда | Описание |
|---------|----------|
| `/caveman` или `/caveman help` | Показать справку |
| `/caveman on` или `/caveman full` | Включить **полный** режим (по умолчанию) |
| `/caveman lite` | Включить **lite** — профессионально, без воды |
| `/caveman ultra` | Включить **ultra** — телеграфный стиль |
| `/caveman wenyan` | Включить **文言文** — классический китайский |
| `/caveman wenyan-lite` | Включить **wenyan-lite** |
| `/caveman wenyan-ultra` | Включить **wenyan-ultra** |
| `/caveman off` | Выключить caveman, обычный режим |
| `/caveman status` | Показать текущий режим |

### Примеры

```
Вы: /caveman full
Бот: 🪨 Caveman режим FULL ВКЛЮЧЁН.

Вы: Как исправить React re-render?
Бот: Новая ссылка объекта каждый рендер. useMemo. Готово.
```

## Режимы

| Режим | Стиль | Экономия токенов |
|-------|-------|------------------|
| **lite** | Деловой, без воды | ~30% |
| **full** | Полный caveman. Фрагменты, без артиклей | ~65% |
| **ultra** | Телеграфный, максимум сокращений | ~75% |
| **wenyan** | Классический китайский стиль | зависит от языка |

---

## Интернационализация

Плагин автоматически использует настройку `bot_language` AstrBot.

Все строки для пользователя (справка, статус, ошибки) локализованы через `.astrbot-plugin/i18n/*.json`. Промпты для сжатия LLM остаются на английском для максимальной совместимости с моделями.

Чтобы добавить новый язык, создайте JSON-файл в `.astrbot-plugin/i18n/` (например, `ja-JP.json`) и перезагрузите плагин. Отсутствующие ключи автоматически берутся из китайского (`zh-CN`).
