# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-05-11

### Added
- Initial release with 6 caveman compression modes: lite, full, ultra, wenyan, wenyan-lite, wenyan-ultra
- Per-session state persistence
- i18n support with 3 languages: zh-CN (fallback), en-US, ru-RU
- Safe system-prompt injection with `[CAVEMAN_MODE_ACTIVE]` deduplication marker
- Trilingual README (English, Russian, Chinese)
